Beautiful color, image, and button variation swatches for WooCommerce product attributes.

== Description ==
Variation Swatches Lite+ replaces default dropdowns with modern, touch-friendly swatches on product pages and archive/shop pages.
It supports color, image, button, and radio styles with tooltip, out-of-stock behavior, and flexible styling controls.

Key features:
- Color, image, button, and radio swatches
- Tooltips (text/image)
- Out-of-stock behavior (hide/blur/disable)
- Archive/shop swatches with alignment control
- Size and font controls for swatches
- Optional default stylesheet
- Shortcode: [wvs_show_archive_variation product_id="ID"]

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/.
2. Activate Variation Swatches Lite+ in Plugins.
3. Configure settings under Variation Swatches.
